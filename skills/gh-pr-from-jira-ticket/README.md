# GH PR From Jira Ticket

This skill opens GitHub pull requests using a Jira ticket key, Jira-derived title, and a consistent PR body template.

## Recommended Scope

`Global/personal skill`

Recommendation: install this globally when you regularly work in repos that use the same Jira plus GitHub PR workflow. It is a reusable authoring pattern rather than repo-specific implementation knowledge.

Invoke it with `$gh-pr-from-jira-ticket`.
