# Sitka Endpoint Layered Architecture

This skill builds and refactors endpoints in `sitka-seafood-market-vercel-functions` using a strict route, controller, service, utils, and docs layering model.

## Recommended Scope

`Project skill`

Recommendation: keep this with the Sitka project. It is tightly coupled to that repository's architecture, file layout, utility conventions, and endpoint validation workflow.

Invoke it with `$sitka-endpoint-layered-architecture`.
